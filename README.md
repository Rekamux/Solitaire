Solitaire by Axel Schumacher
============================

Here is a small implementation of the Solitaire game in Qt, C++.

How to play
-----------

- Install Qt
- Run qmake && make -j8 && ./Solitaire

Features
--------

- Can change cards background
- Can save options and scores
- Smooth animations
- English and french supported
